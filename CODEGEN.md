# Codegen

This repository does not currently generate Rust or Python source as part of the default build.

Any future generated artifacts must:

- Be reproducible from committed inputs.
- Preserve the public API in `ubu_planning_core`.
- Avoid GPU, Python, network, and cloud requirements in default Rust builds.
- Keep committed Cargo manifests on git dependencies pinned by explicit `rev`.

Local dependency overrides belong in `.cargo/config.toml`, which is intentionally gitignored and expected to be generated by `ubu-devshell`.
